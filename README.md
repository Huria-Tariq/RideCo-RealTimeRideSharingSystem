#  RideCo вҖ” Real-Time Ride Sharing System

A frontend web application for a real-time ride-sharing platform built with plain HTML, CSS, and JavaScript вҖ” no frameworks, no build tools. Designed with a 3-role system for Passengers, Drivers, and Admins, and structured for easy backend integration.

> вҡ пёҸ **Note:** This is a frontend-only project currently. Backend integration is in progress.

- рҹ§‘вҖҚрҹ’ј **Passenger Portal** вҖ” Register, request rides, track in real-time, make payments, view history, and rate drivers
- рҹҡҳ **Driver Portal** вҖ” Accept/decline ride requests, manage active rides, and view earnings
- рҹӣ пёҸ **Admin Panel** вҖ” Monitor live rides, manage drivers, and generate reports
- рҹ“Ў **Backend-Ready Architecture** вҖ” API layer pre-written and ready to connect
